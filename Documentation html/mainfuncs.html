main.companyInfoFromCSV(companyListFile, keyWordList, infoList, saveName=None)[source]¶

A function for scraping information off wikipedia pages. Made for checking information about different companies but should be easily transformed to check other kind of information as well. Saves the information in .csv file using unicode-32 encoding.

Note

The Microsoft Excel doesn’t recognize the utf-32 encoding in the same way it recognizes, for example, utf-8 or ANSI encoded .csv files. Thus the .csv file has to be imported manually to excel through Data -> Import -> From text

Parameters:
  • companyListFile – a path to a .csv -file that contains the list of companies the function goes through and finds information about.
  • keyWordList – Key Word List. A _list_ of containing the keywords as _strings_ that the function ‘searchForKeyWords()’ will look for.
  • infoList – Company Information List. A _list_ containing the kinds of information, e.g. “industry” or “Services” as _strings_ that the function ‘searchForCompanyInfo()’ will look for.
  • saveName – The name of the file where the results will be saved and the path to the file as a @string. If parameter is omitted, the path of companyListFile is used instead. Name of the save will be the name of the companyListFile appended with “_results”
